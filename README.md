# OCASI Theme

The theme can be access on this link: 

https://logicaloutcomes.github.io/ocasi-theme/

we are using github pages (https://pages.github.com/) to serve the page from the repo.

The theme is built on:

 * Bootstrap 3.3.7
 * SASS
 * jQuery

boilerplate: https://github.com/willianjusten/bootstrap-boilerplate


## Development Installation for the theme

First of all, install the dependencies to run this boilerplate.

- [NodeJS](http://nodejs.org/)
- [GulpJS](http://gulpjs.com/)

```
npm install
```

### Run development instance

```
gulp
```
